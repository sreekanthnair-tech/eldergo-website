Behind ElderGo

ElderGo began when our founder couldn't find a single tour operator he'd trust with his own parents. So he built one.

That one question — "would I send my parents on this trip?" — has shaped every tour we design. Every hotel we choose. Every tour manager we hire. From our very first trip to today, the question hasn't changed.

“Every tour we run, I ask myself — would I send my own parents on this? If the answer isn't a confident yes, we don't run it.”

— SREEKANTH NAIR, FOUNDER
